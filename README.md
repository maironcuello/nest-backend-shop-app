## Description
__Backend-Shop-App__

## 1. Clone The Proyect
```
https://github.com/maironcuello/nest-backend-shop-app.git
```
## 2. Install
```
$ yarn install
```

## 3. Clone file
```
.env.example to .env
```
## 4. Database PostgreSQL container up
```
docker-compose up -d
``` 
## 5. Run project   
``` 
$ yarn start:dev
```
## 6. Charger Seed Data
```
http://localhost:3000/api/seed
```
